# Covered Topics

This folder organizes covered DFIR topics into the main topic structure.

## What This Folder Contains

- Topic-by-topic `README.md` summaries
- Topics covered under each main folder
- Key learnings from each topic
- Tools used and commands practiced
- A prerequisite topic structure under `Network Forensics`

## Main Folders Overview

- [Basics of Hard Disks and File Systems](./Basics of Hard Disks and File Systems/README.md): foundational storage and file-system concepts. Total days: 1
- [Windows Data Acquisition](./Windows Data Acquisition/README.md): Windows evidence acquisition and integrity basics. Total days: 3
- [Anti-Forensic Techniques](./Anti-Forensic Techniques/README.md): common attacker anti-forensic methods. Total days: 4
- [Digital Forensics on Linux](./Digital Forensics on Linux/README.md): Linux live response, file systems, and host artifacts. Total days: 5
- [Windows Memory Forensics](./Windows Memory Forensics/README.md): Windows memory acquisition and analysis workflow. Total days: 8
- [Linux Memory Forensics](./Linux Memory Forensics/README.md): Linux memory acquisition and memory analysis practice. Total days: 6
- [Network Forensics](./Network Forensics/README.md): currently contains prerequisite network topics for the main topic

## Notes

- `Network Forensics` exists as a main folder, but the main topic summary has not been started yet.
- The prerequisite network topics are organized inside `Network Forensics`.
- The root `README.md` acts as the index for everything inside `Covered-Topics`.
