## Case Study Insights

### 1. Port Scan Detection

* Detected `nmap.exe` as an indicator of scanning activity
* Used command-line analysis to identify scan behavior
* Recognized internal reconnaissance post-compromise

**Insight:** Scanning tools on a host often indicate attacker presence.

---

### 2. Brute Force Detection

* Observed multiple `SYN_SENT` connections
* Linked network activity to specific processes
* Identified repeated authentication attempts

**Insight:** High-frequency connection attempts suggest brute force attacks.

---

### 3. Credential Dumping Detection

* Identified `mimikatz.exe` in memory
* Analyzed credential dumping commands
* Understood LSASS memory targeting

**Insight:** Credential dumping indicates deep system compromise.

---

## Key Takeaways

* Memory forensics reveals live attacker activity
* Correlating artifacts improves detection accuracy
* Understanding attack flow enhances investigation
