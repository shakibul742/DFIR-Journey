### Commands
- certutil -hashfile memdump.mem SHA256

### Notes
- Dynamic acquisition collects data from a running system (RAM, processes, network), while Copy/Duplicate acquisition creates an exact verified copy of files or disks to preserve evidence integrity.
- Hash values (like SHA256) are used to verify that the evidence file has not been modified.
